# 정태의 지각 🏃‍♂️

쿠키런 스타일의 러닝 게임입니다. 정태가 학교에 지각하지 않도록 도와주세요!

## 🎮 게임 방법

### PC
- **스페이스바**: 점프
- **아래 화살표**: 앉기

### 모바일
- **화면 상단 탭**: 점프
- **화면 하단 탭**: 앉기
- 또는 하단의 점프/앉기 버튼 사용

## ✨ 게임 특징

- 📄 시험지 토스트를 수집하여 점수 획득
- 🪙 코인을 모아 정태를 강화
- ❤️ 체력 시스템
- 🚀 속도 업그레이드
- 📊 등급 시스템 (F ~ S)
- 💾 자동 저장 기능

## 🎯 플레이 방법

1. 메인 메뉴에서 코인으로 정태의 능력치 강화
2. 게임 시작 버튼 클릭
3. 장애물을 피하고 아이템을 수집
4. 획득한 코인으로 더 강력하게 업그레이드!

## 🚀 실행 방법

`index.html` 파일을 브라우저에서 열면 바로 플레이할 수 있습니다.
